BrocadeFW

BrocadeFWは、Pythonで書かれた軽量なウェブアプリケーションフレームワークです。

  • Python 2.7以降
  • WSGI対応
  • 正規表現によるURLマッピング
  • 多言語対応（gettextを使わず、文章単位・テンプレート単位での翻訳に対応）
  • Accept-Languageヘッダを解析して、言語やロケールを自動識別
  • Accept-Charsetヘッダを解析して、出力する文字コードを自動識別（エンコードできない文字は文字参照に置き換え）
  • User-Agentヘッダを解析して、スマートフォン・タブレット・フィーチャーフォン等、デバイスに応じた画面を自動的に選択
  • テンプレートエンジンはMakoを採用
  • テンプレートのミニファイ機能つき
